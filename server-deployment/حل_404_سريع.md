# حل مشكلة 404 Not Found - سريع

## المشكلة
nginx يعطي **404 Not Found** - يعني nginx شغال لكن التطبيق مش شغال

---

## الحل السريع (3 خطوات)

### الخطوة 1: الاتصال بالسيرفر
```bash
ssh root@46.224.42.221
```
**كلمة المرور**: `aiadsham`

### الخطوة 2: تشغيل سكريبت الإصلاح
```bash
cd /opt/matrix-platform/server-deployment
chmod +x fix-404.sh
./fix-404.sh
```

### الخطوة 3: التحقق
```bash
curl https://senorbit.ai/health
```

إذا رجع HTTP 200، المشكلة انحلت! ✅

## إذا لم يعمل

### حل يدوي سريع:

```bash
# 1. تشغيل الخدمات
systemctl start postgresql redis-server nginx

# 2. التحقق من PM2
pm2 status

# 3. إذا PM2 مش شغال
cd /opt/matrix-platform
pm2 start pm2.ecosystem.config.js
pm2 save

# 4. التحقق من التطبيق
curl http://localhost:3000/health

# 5. إعادة تحميل nginx
systemctl reload nginx
```
